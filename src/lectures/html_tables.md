### Таблица

Использовались для верстки веб-страниц пока не табличная верстка не сменилось на блочную с помощью слоев.

* В отрисовке таблицы участвуют следующие теги:
  * [&lt;table&gt;](http://htmlbook.ru/html/table): основной контейнер, содержащий строки и ячейки и отображающий своё содержимое как таблицу.
  * [&lt;tr&gt;](http://htmlbook.ru/html/tr): контейнер для добавления строки в таблицу (table row), расположен непосредственно в блоке `<table>`.
  * [&lt;th&gt;](http://htmlbook.ru/html/th): добавление заголовочной ячейки в таблицу (table head), расположен непосредственно в блоке `<tr>`.
  * [&lt;td&gt;](http://htmlbook.ru/html/td) добавление ячейки в строку (table row), расположен непосредственно в блоке `<tr>`.

Таблица с видимой границей и одной строкой шапки:
```html
<table border="1">
  <tr>
    <th>Ячейка 1</th>
    <th>Ячейка 2</th>
  </tr>
  <tr>
    <td>Ячейка 1</td>
    <td>Ячейка 2</td>
  </tr>
</table>
```

### Дополнительные теги для таблиц

Внутри `<table>` допустимо использовать следующие элементы: `<caption>`, `<col>`, `<colgroup>`, `<tbody>`, `<td>`, `<tfoot>`, `<th>`, `<thead>` и `<tr>`.

* В отрисовке таблицы участвуют следующие теги:

  * [&lt;thead&gt;](http://htmlbook.ru/html/thead): Элемент для хранения заголовочных строк вверху таблицы. Допустимо использовать не более одного элемента `<thead>` в пределах одной таблицы, и он должен идти в исходном коде сразу после тега `<table>`.
  * [&lt;tfoot&gt;](http://htmlbook.ru/html/tfoot): Элемент для хранения строк, которые представлены внизу таблицы. Хотя тег `<tfoot>` в исходном коде должен быть определен до тега `<tbody>`, браузеры отображают его в самом низу таблицы.
  * [&lt;tbody&gt;](http://htmlbook.ru/html/tbody): Элемент для хранения строк таблицы, позволяющий создавать структурные блоки, к которым можно применять единое оформление через стили, а также управлять их видом через скрипты.

Таблица с дополнительными блоками:
```html
<table>
 <thead>
  <tr>
   <th> Заголовок </th>
  </tr>
 </thead>
 <tfoot>
  <tr>
   <td> Содержимое футера </td>
  </tr>
 </tfoot>
 <tbody>
  <tr>
   <td> Содержимое ячейки </td>
  </tr>
</table>
```

### favicon

* Иконка, которая отображается на вкладке рядом с заголовком страницы (тег `<title>` в блоке `<head>`)
* Как добавить:
  * Найти нужную иконку
  * Скопировать ссылку либо скачать себе
  * Разместить ссылку на неё в блок `<head>`

```html
<head>
  <title>Заголовок страницы</title>
  <link rel="shortcut icon" href="./favicon.ico">
</head>
```
