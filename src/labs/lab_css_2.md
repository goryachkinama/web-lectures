Написать стили для веб-приложения
* Навигация должна перемещать на соответствующий блок страницы
* Блоки должны быть отцентрированы вертикально
* В одном из блоков использовать картинку в качестве background
* При наведении на картинки/карточки/кнопки срабатывает анимация (сдвигаются, появляется тень, подсвечиваются или перекрашиваются кнопки, масштабируется картинка)
