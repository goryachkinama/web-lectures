### 2. CSS

css

* Написать стили для веб-приложения (раскрасить в указанные цвета и подключить шрифты, например бесплатные от GoogleFonts).
* Постараться отцентрировать весь блок горизонтально и вертикально.
* Будет плюсом, если сам список можно будет прокручивать без прокручивания самой страницы.
* Зафиксировать футер статично внизу, чтобы при уменьшении размера окна он оставался на месте, а менялся лишь размер средней части со списком (заголовок и добавление элементов тоже было на месте).
* Добиться, чтобы при сужении окна кнопки "Добавить" и "Очистить всё" уходили под поле ввода текста.
* То же самое нужно будет проделать с режимами фильтрации, которые пригодятся позже.
* Опционально: сверстать так, чтобы при отсутствии элементов и выводе фразы "Не найдено ни одного дела" строго по центру страница корректно ресайзилась, заголовки и футер не уезжали
* Опционально: добавить фоновое изображение и адаптировать его в зависимости от разрешения (медийные запросы о параметрах устройства),
* Опционально: сделать составной заголовок (картинка + текст) и проконтролировать, что средняя часть не наезжает на верхнюю при изменении размеров экрана
* Опционально: css-модули, задание цветов и размеров через переменные, древовидная структура (scss)
* В процессе можно пользоваться [презентацией](https://dmitryweiner.github.io/web-lectures/Basic%20-%20CSS.html#/).
* Результат коммитить в тот же проект, что и предыдущее задание.
